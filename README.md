# BEE2041 Empirical Project

## Overview
This project investigates the relationship between interest rates and house prices in the UK using data analysis techniques.

## Data
Two datasets are used:
- UK house price data (quarterly)
- UK interest rate data (converted to quarterly averages)

Both datasets are stored in the 'data/' folder.

## Methodology
The data was cleaned and transformed to ensure consistency. Interest rates were aggregated into quarterly averages to align with house price data. The datasets were then merged and analysed using summary statistics, visualisations, correlation analysis, and regression modelling.

## Repository Structure
- 'blog.ipynb' - Main notebook containing analysis, code, and explanations
- 'data/' - Raw and cleaned datasets
- 'scripts/' - Data processing and cleaning script
- 'output/' - Saved graphs and tables

## How to Run
1. Run the script in 'scripts/data_processing.py' to generate the cleaned dataset
2. Open 'blog.ipynb'
3. Run all cells to reproduce the analysis and results

## Results
The analysis finds a negative relationship between interest rates and house prices, supported by both visualisations and regression results.
